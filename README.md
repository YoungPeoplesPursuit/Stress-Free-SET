# Stress Free SET

## A SET clone that makes learning SET easy

SET is a card game where people aim to find SETs, a collection of 3 cards that satisfy certain properties. There are four attributes (number, color, shape, shading) and a SET satisfies that the each attribute is either the same or different across all 3 cards.\
An introduction to the rules and history of SET can be found [here](https://en.wikipedia.org/wiki/Set_(card_game)). 

## Gameplay

If you click 3 cards and they form a SET, the cards will automatically be replaced and a popup box will explain why the SET was valid. Click the popup box to continue playing.\
If you click 3 cards and they do not form a SET, a popup box will appear to explain why it violates the conditions for being a SET. You have the option to try again with the current cards onscreen or press space to replace those 3 cards with new cards. \
The game does not have an obvious ending screen. You can exit the program when there are few cards (6 to 12) remaining and you cannot find a SET.
